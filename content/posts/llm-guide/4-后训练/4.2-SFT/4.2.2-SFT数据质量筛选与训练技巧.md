# SFT 数据质量筛选与训练技巧

> 本文介绍 SFT 阶段的数据质量筛选方法和训练技巧，包括长度筛选、重复率检测、语言质量评分等实践方案. 

---

## 1. 数据质量筛选

### 1.1 长度筛选

基础但有效的筛选条件：
- **instruction 长度** < 10：指令过短，信息量不足
- **output 长度** < 20：输出过短，难以判断质量

### 1.2 重复率检测

计算 n-gram 重复率，过滤低质量样本：

```python
def calculate_repetition_rate(text):
    sentences = sent_tokenize(text)
    if len(sentences) < 3:
        return 0
    
    ngrams = []
    for n in range(2, 5):
        for sentence in sentences:
            words = word_tokenize(sentence.lower())
            ngrams.extend([' '.join(words[i:i+n]) for i in range(len(words)-n+1)])
    
    ngram_counts = Counter(ngrams)
    repeated_ngrams = sum(1 for count in ngram_counts.values() if count > 1)
    
    return repeated_ngrams / len(ngram_counts) if ngram_counts else 0
```

**阈值建议**：重复率 > 0.3 的样本予以过滤. 

### 1.3 语言质量评分

使用小型语言模型对输出文本进行质量评分：
- 语法正确性
- 语义连贯性
- 风格一致性

## 2. 训练配置

### 2.1 基础配置

| 参数 | 推荐值 | 说明 |
|:-----|:-------|:-----|
| 学习率 | 2e-5 | SFT 典型范围 |
| Batch Size | 4 | 单卡 |
| 梯度累积 | 8 | 等效 batch size = 32 |
| 训练轮数 | 3 | 避免过拟合 |
| Weight Decay | 0.01 | 正则化 |
| Warmup Ratio | 0.1 | 稳定初期训练 |

## 3. 高级技巧

### 3.1 多轮对话数据构造

- 保持对话历史的连贯性
- 控制每轮回复长度
- 添加系统提示词模板

### 3.2 数据混合策略

- 通用能力数据 + 领域专业数据
- 简单任务 + 复杂任务混合
- 控制各类别比例

### 3.3 动态采样

根据训练过程中的 loss 变化动态调整采样权重：
- 高 loss 类别增加采样概率
- 低 loss 类别降低采样概率

---

## 4. 常见问题

### 4.1 过拟合

- **现象**：训练 loss 持续下降，验证 loss 上升
- **解决**：减少训练轮数、增加 dropout、数据增强

### 4.2 灾难性遗忘

- **现象**：通用能力下降，仅保留微调任务能力
- **解决**：混合原始预训练数据、使用 LoRA 而非全参数微调

### 4.3 长度偏差

- **现象**：模型倾向于生成过长或过短的回复
- **解决**：调整长度惩罚、数据长度归一化

---

## 5. 总结

SFT 的核心是**数据质量**而非数据数量：
1. 严格筛选低质量样本
2. 控制数据分布平衡
3. 合理设置训练超参数
4. 持续监控验证指标
