## 关于我

我是 **SingularGuyLeBorn**，一名专注于 **AI 应用工程与知识管理** 的独立开发者。

日常工作围绕三件事展开：读论文与源码、把理解写成 Markdown、用 Agent 把知识变成可执行的 workflow。KnowPilot 是我把「博客 + 数字花园 + Agent 工作台」收拢到一个 monorepo 里的长期项目。

## 现在在做什么

- 打磨 **KnowPilot** L2 Agent 能力：ReAct 循环、Skill/MCP 双路径、流式 Chat、上下文压缩
- 维护个人知识库：LLM 学习笔记、Transformer 数学推导、Web 基础与 AI 时代个人成长反思
- 参考 **Claude Code / OpenClaw / Codex** 等优秀 Agent 产品在 UI、UX、token 控制上的设计，持续改进交互

## 技术偏好

- **UI**：莫兰迪暖色 + 玻璃拟态 + 弹簧动效（150–300ms），图标只用 Lucide/SVG
- **架构**：单文件收拢（router / services / hooks / shared UI），避免目录爆炸
- **Agent**：`native:` / `skill:` / `mcp:` 三类工具统一桥接；只读并发、写入串行
