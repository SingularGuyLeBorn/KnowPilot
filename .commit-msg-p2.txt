feat(swarm): Phase 2 — run_async 统一 subagent session + task_status/await_async/cancel_async 工具

异步任务可视化(为 Phase 3 UI 铺路):
- startAsyncAgentTask 额外创建 ChatSession(kind=subagent, parentSessionId,
  taskDescription, status=running)作为任务可视化载体,关联 jobId
- buildAsyncExecute 任务完成/失败时同步 subagent session status(completed/failed)

新增 native 工具:
- task_status: 查单个/列表异步任务状态(queued/running/completed/failed + elapsedMs +
  subagentSessionId),来源 orchestrator.isRunning/isQueued + Task 表
- cancel_async: 取消运行中/排队中任务(复用 cancelAsyncJob)
- await_async: 显式阻塞等待任务结束返回结果

run_async 增强:
- 新增 waitForResult 参数:true=阻塞等待结果直接返回(runAsyncTool 调
  waitForAsyncJob 轮询 Task 完成态);false(默认)=立即返回 running 进队列
- waitForAsyncJob 最长轮询 10 分钟(受工具超时 race 兜底)

asyncJobOrchestrator 新增 isRunning/isQueued 查询方法
AsyncTaskInput 新增 subagentSessionId 字段

验证: tsc 0 错误; server vitest 228 passed(更新 ALL_NATIVE_TOOL_NAMES 含 3 新工具)
